CSS3 Box-sizing Property

Jimmy Xie
27 OCT 2017

CSS3 has a new property named box-sizing, its value can be content-box(default), border-box and inherit.


        box-sizing: content-box || border-box || inherit

content-box

    Default value. Using content-box keep the W3C standard, which means element width/height = border + padding + content width/height

border-box

    Using this value means element width/height equals to content width/height, and the border and padding are inside. element width/height = content width/height

inherit

    As the name meaning, it will inherit it parent's box-sizing property.

Sometimes it is confused to use them. Following are some example which help me to use them.

            .container .box {
                width: 200px;
                height: 200px;
                background-color: darkgreen;
                padding: 10px;
                border: 5px solid black;
                text-align: center;
                line-height: 200px;
                color: #f5f5f5;
                margin: 33px auto;
            }

            .container .content-box {
                -webkit-box-sizing: content-box;
                -moz-box-sizing: content-box;
                box-sizing: content-box;
            }

            .container .border-box {
                -webkit-box-sizing: content-box;
                -moz-box-sizing: content-box;
                box-sizing: border-box;
            }

            .container .inherit {
                -webkit-box-sizing: inherit;
                -moz-box-sizing: inherit;
                box-sizing: inherit;
            }
